# Omnia
A simple file explorer made with React + Rust (Tauri 2.0)
<img width="900" alt="image" src="https://github.com/user-attachments/assets/a57f197c-9134-4210-9aa7-ed4ea547c4c2" />

### What to expect?
A really early stage project without some core features and subtle to several design changes

### Early implemented features
* User folder as initial path
* Double click folders to access
* Navigation through breadcrumb
* Navigation through text input

### What's next ?
* Open files
* Copy/Paste files
* Select multiple files
* Properties with folder/file metadata
